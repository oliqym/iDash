Submission:
Each participating should submit the solution in two containers based on official Ubuntu docker image, of which each can be run on an independent server running Ubuntu. The containers should set up a communication channel to exchange the (protected) input and other necessary data. Each team is expected to also submit the source code of their solution. The source code will only be used for evaluation purpose (i.e., checking the security compliance) by the organizers.
